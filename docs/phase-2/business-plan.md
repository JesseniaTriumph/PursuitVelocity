# Business Plan
## Pursuit Sync / Velocity

---

## CEO Perspective: Vision & Strategy

### Mission
Give every Pursuit fellow the infrastructure to be discovered, collaborate, and ship real products — regardless of their background.

### Vision (3-year)
Velocity becomes the default professional layer for 50,000 early-career developers from equity-focused programs nationwide. It is where builders are found, where teams form, and where careers are launched.

### Strategy
1. **Win the captive network first.** Pursuit's 2,000-person network is the beachhead. No cold start. No chicken-and-egg.
2. **AI as the moat.** Campfire's matching algorithm and autonomous agents give us a differentiation no static directory can replicate.
3. **Revenue through employers, not ads.** Employers pay to find vetted, pre-qualified talent. Builders never pay.
4. **Expand via network effects.** Each new fellow makes the platform more valuable for existing members.

### 12-Month Milestones
| Month | Milestone |
|-------|-----------|
| M1 | Launch to all active Pursuit fellows (200 users) |
| M3 | 500 MAU; Campfire match rate ≥ 25% |
| M6 | GitHub integration live; agent layer running |
| M9 | iOS + Android in TestFlight |
| M12 | 3,000 MAU; $5k MRR from 2 employer partners |

## CFO Perspective: Financial Model

### Revenue Model
**Primary: Employer Partner Subscriptions**
- Price: $199/mo per hiring manager seat
- Target: 3 employers by M9, 10 by M18
- Projected ARR (M18): $23,880 (10 employers)

**Secondary: Institutional License (Pursuit)**
- Price: $12,000/year flat fee for unlimited fellows
- Projected ARR: $12,000 at M12

**Tertiary: Grant Funding**
- Target: 1 tech equity grant ($50k–$100k) in Year 1
- Probability: 40% (strong mission alignment)

### Cost Structure (Year 1)
| Category | Monthly | Annual |
|----------|---------|--------|
| Infrastructure (Phase 1) | $70 | $840 |
| Infrastructure (Phase 2+) | $300 | $2,100 |
| Claude API | $50 | $600 |
| Total Year 1 | ~$350 avg | ~$3,540 |

### Runway
- **Without revenue:** $3,540/year = sustainable indefinitely on bootstrap
- **With grant ($50k):** 14+ years of runway at Phase 2 burn rates
- **First revenue needed for scale:** Month 9 (employer portal launch)

## CMO Perspective: Marketing & Growth

### Target Channels
1. **Pursuit program integration** — mandatory for all fellows (zero CAC)
2. **Lookbook SEO** — every builder's public page indexed by Google
3. **GitHub README links** — builders link Velocity profile from GitHub bio
4. **LinkedIn posts** — builders share Lookbook URL when job searching
5. **Events** — hackathons, demo days as acquisition events
6. **Staff + mentor amplification** — Pursuit staff share on LinkedIn/X

### Brand Voice
- **Direct:** No buzzwords. Say what it does.
- **Warm:** This is a community, not a product.
- **Empowering:** Every word should make the builder feel capable.

### Key Messages
1. "Your Lookbook is your story. Tell it in public."
2. "The algorithm finds who you should meet. You decide if you want to."
3. "Built by Pursuit fellows. For Pursuit fellows."

### Growth Metrics
| Metric | M3 Target | M12 Target |
|--------|-----------|-----------|
| MAU | 500 | 3,000 |
| Profile completion rate | 70% | 85% |
| Campfire match → message rate | 20% | 35% |
| Lookbook unique views/mo | 1,000 | 15,000 |
| Event RSVPs/mo | 200 | 1,500 |
